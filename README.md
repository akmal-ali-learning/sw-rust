# sw-rust
Learning rust by solving problems.

Aims:
To learn rust by solving all 189 problems in the Cracking the Code Interview 6th edition book.

How to use:
cd solutions

Each problem is a seperate lib crate.
To check functionality run the tests inside each crate.

cargo test problem_x_test
